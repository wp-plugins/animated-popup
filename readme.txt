== Description ==
Add a pop-up to your site to get new subscribers. Mailing list marketing is the best way to grow your business.

== Installation ==

1. Download from Wordpress
2. Activate on the plugins page. Click 'Settings' on the plugins page or 'Animated Popup' in the sidebar to modify settings.
3. Place the widget somewhere on your site or use the shortcode-- [animated_popup]

== Frequently Asked Questions ==

= Why isn't the pop-up showing up? =

You need to add a widget or shortcode before the pop-up will display.

= Why isn't the shortcode working? =

Make sure that your shortcode has an underscore, not a dash, and looks like this [animated_popup]

= Why is the pop-up stuck in the sidebar like a normal widget? =

The pop-up is in "Embed" mode. Go to the Settings page and switch it back to pop-up mode.

= How do I add subscribers to my mailing list automatically? =

To do this, you'll need to activate the [Pro version](http://aklosismedia.com/quotes/dev/downloads/animated-popup) of the plugin.
